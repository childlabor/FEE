React重写了一个之前项目的系统框架，并用webpack打包
--------------
###### 技术栈：
- `React`
- `Webpack`
- `jQuery`

###### 测试说明：
1. 下载文件包至本地
2. `cmd`进入文件夹, `npm install` 安装依赖包
3. `npm test` 运行程序
4. 浏览器打开`http://localhost:8080/dist/test.html` 查看页面；  
  `http://localhost:8080` 可查看各文件源码  


引入了热更新模块，直接修改代码，可实时在浏览器下查看，不需重复运行 `npm test`
